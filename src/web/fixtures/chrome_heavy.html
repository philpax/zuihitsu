The Lumen Ledger — Project Overview

The Lumen Ledger is an open-source double-entry bookkeeping library written in Rust. It models every transaction as a balanced set of postings, so the books can never drift out of balance.

Its core is a deterministic ledger engine: given the same sequence of journal entries, it always produces the same account balances, which makes the library easy to test and to audit.

The project ships a command-line importer that reads bank statements in CSV and maps them onto accounts using a small set of user-defined rules.
