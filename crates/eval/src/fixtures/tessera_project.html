Tessera

Tessera is a content-addressed file storage library for Rust. It stores every file as a set of variable-sized chunks, addressing each chunk by the hash of its contents, so two files that share data store that data only once. It is built for backup tools and build caches, where the same bytes recur across many versions of the same tree.

Chunking is done with a rolling hash that splits a byte stream at content-defined boundaries, targeting an average chunk size of 64 KiB. Because the boundaries follow the content rather than a fixed offset, inserting a few bytes near the start of a file re-chunks only the region around the edit, leaving every later chunk — and its stored copy — untouched.

Each chunk is named by its BLAKE3 hash, and a file is recorded as an ordered list of those chunk hashes. Retrieving a file means looking up each chunk by hash and concatenating them. Identical chunks collapse to one stored object, so a repository of near-identical trees stays close to the size of a single tree.

Tessera is written in pure Rust with no unsafe code, is released under the MIT license, and is maintained by the quill-labs collective. The store is a single append-only file plus an index, so a backup can be copied with an ordinary file copy.
